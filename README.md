# Snack-Delivery-Web-Application

A full-stack snacks delivery web application developed using Django, allowing users to browse snacks, add products to cart, place orders, and manage deliveries through a responsive and user-friendly interface.

Features
Snack browsing and product listing
Add to cart and checkout system
Order management functionality
Responsive modern UI
Backend data handling with Django
Secure form and database integration
Tech Stack
Python
Django
HTML5
CSS3
JavaScript
SQL / SQLite
Project Highlights
Developed complete frontend and backend architecture
Implemented dynamic product and order management
Built mobile-responsive layouts for better user experience
Designed scalable structure for future payment and tracking integrations
Purpose

This project was created to demonstrate full-stack web development skills, backend integration, and responsive e-commerce application design using Django.
